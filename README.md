# InstaNameGen ⚡

AI-powered name generator for baby names, business names, pet names, fantasy characters, usernames, and band names.

## Stack
- React + Vite + Tailwind CSS
- Claude Haiku API (name generation)
- Google AdSense (monetization)
- Vercel (hosting)

## Setup

```bash
npm install
cp .env.example .env
# Fill in your API keys in .env
npm run dev
```

## Environment Variables

See `.env.example` for required variables.

## Deploy

Push to GitHub → auto-deploys on Vercel. Add env vars in Vercel dashboard.
